About Me

Hi, I'm Daniel McCluskey, a Graduate Computer Games Programmer, working as a Research Assistant for the Creative Computing Courses at the University of Gloucestershire, currently focusing on writing course material for Artificial Intelligence, Data Science and Games Programming related courses.

Skilled with the Unity engine for mobile and PC development using C#, Unreal Engine development for games using C++, and experience with the programming API and Engine integrations of the FMOD audio framework. I also have a keen interest in Artificial Intelligence and Machine Learning for games and have gained experience in AI techniques including behaviour trees, state machines, goal-oriented action planners, pathfinding algorithms and basic Neural Network implementations for games during my time at University as both a student and a Research Assistant.

Self-taught knowledge of website development using HTML and CSS, using frameworks such as Bootstrap and Skeleton. Experience with using Python for data science in tandem with libraries such as FastAI, Pandas, Matplotlib and Numpy. Knowledge of ECS through practical implementation in C++ and Amateur experience with OpenGL and GLSL shader programming in C++.
